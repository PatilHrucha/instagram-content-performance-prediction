# Instagram Content Performance Prediction

## Project Overview

This project predicts the performance of an Instagram post using Machine Learning.

The model classifies posts into one of four categories:

- Low
- Medium
- High
- Viral

## Algorithm Used

Random Forest Classifier

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## Dataset

Instagram Analytics Dataset

## Features

- Account Type
- Media Type
- Content Category
- Traffic Source
- Follower Count
- Likes
- Comments
- Shares
- Saves
- Reach
- Impressions
- Engagement Rate
- Followers Gained
- Caption Length
- Hashtags Count

## Model Accuracy

85.13%

streamlit run app.py
